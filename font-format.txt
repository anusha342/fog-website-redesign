| Display     | "Neue Haas"   | Arial, sans-serif          | Google  |
| Body        | "Inter"       | Helvetica, sans-serif      | Google  |

## Type Scale

| Token     | Element   | Font Role | Size (rem) | Weight | Line Height | Letter Spacing | CSS Variable          |
|-----------|-----------|-----------|------------|--------|-------------|----------------|-----------------------|
| `heading-1` | `<h1>`  | Display   | 3.5        | 700    | 1.1         | -0.02em        | `--fs-heading-1`      |
| `body-base` | `<p>`   | Body      | 1          | 400    | 1.6         | 0              | `--fs-body-base`      |

## Weights in Use

| Weight Name | Value | Used In              |
|-------------|-------|----------------------|
| Regular     | 400   | Body, captions       |
| Medium      | 500   | Labels, UI elements  |
| Bold        | 700   | Headings, CTAs       |

## Usage Rules

- `Display` font is **only** used for headings (h1–h3) and hero text
- `Body` font is used for all UI text: paragraphs, labels, buttons, nav
- Never apply `heading-1` or `heading-2` on viewport < 768px — use `heading-3` instead
- Minimum body text size: `0.875rem` (14px)
- Do not mix weights within a single line of text
